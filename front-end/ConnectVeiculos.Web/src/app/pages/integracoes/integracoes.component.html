@if (gmAbaTutorial) {

Voce precisa fazer no Google antes de plugar aqui:

  1. Crie uma conta no Google Merchant Center e anote o Merchant ID (canto superior).
  2. Acesse Google Cloud Console e crie um projeto.
  3. Em APIs e Servicos > Biblioteca, ative a Content API for Shopping.
  4. Em APIs e Servicos > Credenciais, crie credenciais tipo OAuth 2.0 Client ID (tipo "Desktop app"). Copie o Client ID e o Client Secret.
  5. Use o OAuth Playground para gerar o Refresh Token:
    • No icone de engrenagem, marque "Use your own OAuth credentials" e cole Client ID + Secret.
    • Em Step 1, selecione o escopo https://www.googleapis.com/auth/content.
    • Autorize com a conta Google que tem acesso ao Merchant Center.
    • Em Step 2, clique em Exchange authorization code for tokens e copie o refresh_token.
  6. Cole tudo na proxima aba e clique em Testar.

warning O Refresh Token nao expira (a menos que seja revogado manualmente). Guarde com cuidado. Detalhes em GUIA-INTEGRACOES.md > secao 3.
